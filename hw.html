This assignment has two parts:

(a) First, modify HW1 so that the regression is used to explain "return deviations" of all the assets using the "return deviations" of the 10 assets you select. This should entail a straightforward modification of your script, however the results may be different.

(b)  Next, build a factor model using the regression coefficients from the winning set of ten assets. (Factor models are explained in the first set of documents under Lecture 5). So if you are looking at n assets in total, you obtain an 10 x n matrix (each column has the regression coefficients). This is the "V" matrix. The "F" matrix is the covariance matrix of the asset returns for the ten assets. And the "sigma squared" coefficients, or idiosyncratic variances, are simply the part of of each stock's return variance that is not accounted for by the V^T F V matrix.
